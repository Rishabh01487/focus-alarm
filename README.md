# Focus Alarm Pro

Focus Alarm Pro is a web-based productivity tool designed to help you set goals, stay focused, and get reminders when you forget.

## Features

- **Goal Setting**: Define specific goals with start and end times.
- **Smart Reminders**: Audio and visual notifications when goals are overdue or time is running out.
- **Progress Tracking**: Visual progress bars for active goals.
- **Statistics**: Track total, active, completed, and overdue goals.
- **Customizable**: Choose different alarm sounds and reminder frequencies.
- **PWA Support**: Installable as a Progressive Web App on mobile and desktop.

## How to Use

1. Open `index.html` in your web browser.
2. Click "Add New Goal" to set up your tasks.
3. Enable notifications for the best experience.

## Technologies

- HTML5, CSS3, JavaScript (Vanilla)
- LocalStorage for data persistence
